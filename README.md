# 📄 AI Resume Analysis System

An AI-powered Resume Analysis Web Application built using **Python, Streamlit, Machine Learning, Regex, and Data Visualization**.

This project allows users to upload resume files (PDF/CSV), extract resume content, predict job category, detect skills, generate resume scores, and visualize top keywords.

---

## 🚀 Features

- Upload Resume in PDF/CSV format
- Extract text from resumes
- Predict Job Category using Machine Learning
- Detect important technical skills
- Generate Resume Score
- Visualize top resume keywords
- Interactive UI using Streamlit

---

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-learn
- Regex
- Matplotlib
- pdfplumber
